#### picker

*************
这是一个仿微信小程序的```pikcer```组件，滚动选择器，现支持三种选择器，通过mode来区分，分别是普通选择器，时间选择器，日期选择器，默认是普通选择器。

************
##### 使用:
安装：
```
npm install 
```
运行:
```
npm start
```
***************
##### OPTIONS

普通选择器：```mode = selector```

属性  | 类型 | 默认值 | 说明
---|---|---|---
row 1 col 1 | row 1 col 2
row 2 col 1 | row 2 col 2
